#ZooDirectory_AndroidMobileApp

- This is a fictional zoo android mobile app that consists of listing of animals,
  details about each animal and information about the zoo.
- The first activity: listing of animals uses a thumbnail image of the animal along with
  the name of the animal that is implemented using listview. When clicked, takes to the 
  second activity.
- The animal detail activity shows the details of the animal that was clicked.
- The zoo information activity provides the basic information of the zoo. 
- Call intent feature is used to dial the phone number present in the app.
- Menu items has an option to uninstall the app.
